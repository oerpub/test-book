Undo Expectations 

1. Basically what I expect to happen when I hit undo, is that things undo up to the last structural change. If I hit undo anytime in this paragraph, some subset of what I typed should disappear, but not the heading above me or the paragraph above (if there were one) or the semantic element above me.

2. Inside this subheading hitting undo should only delete some part of this heading, not the para before. 

4. Hitting undo while in the title, should only delete up to the title.

3. Right after I add this note, undo should just remove the note. If I hit undo after typing some in this para inside the note, only characters inside this para should disappear and maybe this whole para, but not more.

5. If I delete this note and hit undo, the entire note should restore, and nothing else should happen.

6. If I hit undo and then redo, my document should look like it did just before I hit undo.

If I select "Copy quotation" and then hit undo, it should undo something I did before that action. Since copy doesn't change anything.
